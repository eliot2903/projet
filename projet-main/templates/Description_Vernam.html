Vernam

Le chiffrement de Vernam est une méthode de cryptographie qui consiste à combiner un message avec une clé secrète unique et aléatoire.

Chaque caractère du message est transformé à l’aide d’une opération logique (généralement le XOR) appliquée avec le caractère correspondant de la clé, produisant ainsi un texte chiffré.

Lorsque la clé est aléatoire, utilisée une seule fois et strictement confidentielle, cette méthode devient un « masque jetable » (one-time pad), offrant un niveau de sécurité maximal.

Pour déchiffrer le message, il suffit d’appliquer la même opération avec la même clé, ce qui permet de retrouver le texte original.
