Chiffre de Trithémius

Le chiffre de Trithémius est une méthode de cryptage polyalphabétique qui repose sur un décalage progressif des lettres du message. Chaque caractère est chiffré en utilisant un décalage croissant : la première lettre est décalée de 0, la deuxième de 1, la troisième de 2, et ainsi de suite.

Ce procédé utilise une table appelée tabula recta, composée de plusieurs alphabets décalés. À mesure que l’on avance dans le texte, on utilise l’alphabet suivant, ce qui rend le cryptage plus difficile à casser qu’un simple chiffrement de César.

Pour décoder un message chiffré avec la méthode de Trithémius, il suffit d’appliquer les décalages inverses dans l’ordre : la première lettre est décalée de 0 vers l’arrière, la deuxième de 1, la troisième de 2, etc., jusqu’à retrouver le texte original.
